# Running log in test cases

1) Clone project to local directory
    - github link: https://github.com/omewahj/Hudl_Repo.git
    - run in terminal: git clone https://github.com/omewahj/Hudl_Repo.git

2) CD in project and install dependencies from requirement.txt
    - run in terminal:
        cd Hudl_Repo.git
        pip install -r .\requirements.txt

3) Enter password into password.txt file
    - Password will be a base64.b64encoded value (will be provided to you)

4) Run log in test
    - run in terminal:
        python3 Login_Test.py
